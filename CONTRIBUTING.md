# Contributing

We welcome contributions to this project! Here's a quick guide to help you get started.

## Running Tests Locally

To ensure your changes don't introduce regressions, please run the tests before submitting a pull request. This project uses `pytest` for its Python tests. You can install `pytest` if you haven't already (it's usually included in `requirements.txt` or `pyproject.toml` for development dependencies) and then run all tests from the project root using:

```bash
pytest
```
